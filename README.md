# HanTooApp (한국투자증권 실전 자동매매 앱)

안드로이드 환경에서 한국투자증권 Open API를 활용하여 주식 자동매매를 수행하는 어플리케이션입니다. 래리 윌리엄스의 변동성 돌파 전략을 기반으로 하며, 갤럭시 탭 등을 활용한 24시간 상시 서버 모드 동작에 최적화되어 있습니다.

## 🚀 주요 기능

- **24시간 무한 서버 모드**: 앱을 종료하지 않고 켜두면 매일 아침 9시 매매를 시작하고 장 마감 후 대기 모드로 전환됩니다. (익일 오전 8시 50분 자동 초기화)
- **변동성 돌파 실시간 포착**: 
  - 당일 시가 대비 **2% 이상 급등**하기 시작하는 종목을 실시간 상승률 순위에서 자동으로 발굴하여 감시 리스트에 등록합니다.
  - **추격 매수 방지**: 타겟가 돌파는 했지만 이미 너무 올라버린 종목(타겟가 +1.5% 초과)은 매수하지 않고 걸러냅니다.
- **래리 윌리엄스 전략 기반 타겟가**: 
  - `당일 시가 + (전일 고가 - 전일 저가) * 0.5`
- **리스크 관리 및 수익 확정**:
  - **익절**: 매수가 대비 `+3.0%` 도달 시 시장가 매도.
  - **손절**: 매수가 대비 `-1.5%` 하락 시 즉시 손절.
  - **당일 청산**: 오후 `15:20`에 보유한 모든 종목 전량 매도 (오버나이트 금지).
- **자금 관리**: 
  - 최대 5종목 동시 보유, 종목당 100만원 매수 설정 (총 500만원 규모).
- **편의 기능**:
  - 실전 계좌 잔고, 총 매수금액, 총 평가손익 실시간 확인.
  - 종목 클릭 시 네이버 금융 모바일 차트 연동.
  - 상단 알림바를 통한 봇 상태 및 퇴근 보고.

## 🛠️ 설정 방법

### 1. API 키 발급
한국투자증권 KIS Developers 센터에서 **실전매매용** App Key와 App Secret을 발급받아야 합니다.

### 2. 로컬 설정 (`local.properties`)
보안을 위해 API 키와 계좌정보는 Git에 업로드하지 않으며, 프로젝트 루트의 `local.properties` 파일에 아래 내용을 추가합니다.

```properties
hantoo.app.key=발급받은_실전용_앱키
hantoo.app.secret=발급받은_실전용_앱시크릿
hantoo.account.number=계좌번호8자리-01
```

### 3. 태블릿/스마트폰 최적화
안정적인 백그라운드 동작을 위해 아래 설정이 권장됩니다.
- **배터리 최적화 제외**: 설정 > 애플리케이션 > HanTooApp > 배터리 > **'제한 없음'**.
- **절전 모드 끄기**: 시스템 전체 절전 모드 비활성화 권장.
